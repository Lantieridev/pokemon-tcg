Historial cronológico de acciones. Controles interactivos (Play, Pausa, Speed, Retroceso, Avance) para auditar cada evento del match.

¿Cómo probarlo?

  1. Puedes usar el **Modo Simulación** (activado por defecto) para probar la interactividad del chat, simular desconexiones/reconexiones (comprobando el buffer) y usar los controles de replay sin levantar el backend.
  2. O puedes levantar el backend (puerto 8081), registrarte, loguearte para obtener un JWT real, y conectarte al WebSocket WebSocket real para una prueba de punta a punta.
